# SM Tech Solutions — Website (Vite + React)

Stack: **React**, **TypeScript**, **Vite**, **Tailwind CSS**, **React Router**.

## Run locally

```bash
cd sejal-web
npm install
npm run dev
```

Build:

```bash
npm run build
npm run preview
```

## Contact API + MongoDB

Optional: connect **`../sejal-api`** for `POST /api/contact`. See `.env.example` and `vite.config.ts` for proxy / mock settings.

## Deploy (Vercel)

- **Framework preset:** Vite  
- **Build command:** `npm run build`  
- **Output directory:** `dist`
